I will create the **Google Photos** icon in this article using CSS only. Let's look at how we do that.

## Problem

![problem](https://imgur.com/MKql9lI.png)

## Solution

First, we need to create the structure for this logo then we will style that structure.

### HTML 

<CodeTitle lang="html" title="index.html" />
```html
<div class="wrapper">
  <div class="yellow"></div>
  <div class="red"></div>
  <div class="green"></div>
  <div class="blue"></div>
</div>
```

### CSS 

<CodeTitle lang="css" title="style.css" />
```css
.wrapper {
  width: 200px;
  aspect-ratio: 1;
  display: grid;
  grid-template-columns: 1fr 1fr;
}

.yellow, .blue {
  width: 100%;
  height: 55%;
}

.red, .green {
  width: 55%;
  height: 100%;
}

.yellow {
  border-radius: 1in 1in 0 7px;
  place-self: end;
  background: #fbbc01;
}

.red {
  border-radius: 7px 1in 1in 0;
  background: #ea4132;
}

.green {
  border-radius: 1in 0 7px 1in;
  place-self: end;
  background: #069e57;
}
.blue {
  border-radius: 0 7px 1in 1in;
  background: #4086f4;
}
```
